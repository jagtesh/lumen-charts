# Lumen Charts WASM SDK

A safe, idiomatic WebAssembly SDK for [Lumen Charts](https://github.com/jagtesh/lumen-charts), a high-performance, GPU-accelerated charting library inspired by TradingView's Lightweight Charts.

This package provides the main JavaScript/TypeScript API for interacting with Lumen Charts from the browser. It implements an interface that mirrors the Lightweight Charts (LWC) v5 API while leveraging WebGPU and Canvas2D for maximum performance.

Please visit the [GitHub repository](https://github.com/jagtesh/lumen-charts) for full documentation, architectures, and demo examples.
