# Lumen Charts SDK

A safe, idiomatic Rust SDK for [Lumen Charts](https://github.com/jagtesh/lumen-charts), a high-performance, GPU-accelerated charting library inspired by TradingView's Lightweight Charts.

This crate provides the main end-user API for interacting with Lumen Charts from Rust. It implements a fully typed interface that closely mirrors the Lightweight Charts (LWC) v5 API while maintaining zero-cost overhead to the core engine.

## Usage

```toml
[dependencies]
lumen-charts = "2.0.0"
lumen-charts-sdk = "1.0.0"
```

Please visit the [GitHub repository](https://github.com/jagtesh/lumen-charts) for full documentation, architectures, and examples (like `examples/rust-demo`).
